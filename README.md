## What is R3D?

The R3D License is a custom open license designed specifically for three-dimensional rendering libraries. It gives developers the freedom to use, integrate, and distribute 3D software in their own projects — personal, academic, or commercial — while protecting the rights of the original creator.

## What You Can Do

- Use the library in any project, including commercial ones
- Integrate it into your own applications and products
- Redistribute it as part of a finished product
- Contribute improvements back to the original project

## What You Cannot Do

- Sell or sublicense the library as a standalone product
- Remove or hide copyright notices and license identifiers
- Use the creator's name or identity for promotion without written consent
